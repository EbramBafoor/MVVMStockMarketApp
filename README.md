# MVVMStockMarketApp

## Project Features
- CSVParser
- MVVM with Clean Architecture
- Coroutines
- Kotlin Flow
- Navigation Components
- Hilt for Dependency Injection
- Retrofit
- Room Database
- Cashing Data
- Jetpack Compose
- Custom Graph
